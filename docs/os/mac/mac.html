快捷命令

显示全路径的命令：

Finder内置了「显示路径栏」的功能, 并配置了快捷键(Option+Cmd+P).

打开“终端”，输入以下两条命令：

defaults write com.apple.finder _FXShowPosixPathInTitle -bool TRUE;killall Finder

恢复默认状态状态显示命令

defaults delete com.apple.finder _FXShowPosixPathInTitle;killall Finder

mac系统下赋予 shell 文件可执行权限

chmod +x file
chmod +x build.sh

MAC 命令行解压 rar 文件

使用Homebrew安装unrar

brew install unrar

cd到rar文件的目录下，然后在终端输入下列命令：

unrar x 需解压的文件目录

解压7z格式文件

brew search 7z
7z e filename.7z

使用免费软件 The Unarchiver

https://theunarchiver.com

问题

Mac终端 sudo:command not found 解决方法

首先输入命令，让暂时可以使用vim等命令

export PATH=/usr/bin:/usr/sbin:/bin:/sbin:/usr/X11R6/bin

然后打开~/.bash_profile

vim ~/.bash_profile

重启profile

source ~/.bash_profile

mac 挂载移动硬盘

diskutil list

卸载硬盘

sudo umount /dev/disk2s1

重新挂载分区

sudo mount_ntfs -o rw,nobrowse /dev/disk2s1 ~/Desktop/md1

执行完命令后，访问桌面上的 md1 即为访问硬盘分区，现在是可写状态。

系统清理

1、清空回收站。

2、在文稿里，按文件大小排序，删除不需要的文件。

3、对于GarageBand，这个是系统上的模拟乐器，一般都使用不到。

清除方法：

rm -rf /Library/Application\ Support/GarageBand
rm -rf /Library/Application\ Support/Logic
rm -rf /Library/Audio/Apple\ Loops

查看系统的文件占用详情

du -sh *

cd ~/Library
du -sh *

不断去找出暂用存储容量大的文件夹。
